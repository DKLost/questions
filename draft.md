# 关于本软件中知识点的实现

任意一条知识点都被视作一条构造。

一条构造由4部分组成：中心存在，方式存在（构造过程），注入存在，构造结果（意指结果）

- 注1 ⇒ 由于构造结果实则是完全确定的，即存在一个唯一的直接构造结果，本身是不必要明示的，因而后续描述中的构造结果，大多指向非唯一的意指结果。

一条构造可以理解为对中心P以方式F注入存在X，从而构造一个仍是P的结果P'的过程。

对一个知识点的复习，即被视作再引发一条构造，并判断引发结果是否正确的过程。

通常题干-答案式的知识记录，答案本身便是一种对构造结果的引起，而构造过程，即方式、中心、注入三者则被隐含在了题干当中。

为了方便描述，可以将任意一条构造都可以转化描述为：P'是将中心P的Y确定为X的P。其实质是找到一个方式为.Y=的等价构造。

例如：

问题：在水溶液中或熔融状态下能够电离出离子的化合物是什么？。

答案：电解质。

其本身可被视作这样的一条构造：

1. 中心数据：在水溶液中或熔融状态下能够电离出离子的某物
2. 方式存在：确定某物为
3. 注入存在：化合物
4. 意指结果存在：电解质

- 注意

  1. 对于"在水溶液中或熔融状态下能够电离出离子的化合物是电解质"，(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)，是完整表述

  2. "结果存在"本身是自动生成的一种固定结果，并不具有指向其他存在的能力，这里方便描述，使用的是意指后的结果，使其可以指向其他存在，实则是承载了某种"是"的意指存在。即实质是：

     先有：

     1. 中心存在：{}
     2. 方式存在：所是
     3. 注入存在：(<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物))
     4. 意指结果：(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)

     后有：

     1. 中心存在：inject
     2. 方式存在：自
     3. 注入存在：(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)
     4. 意指结果：电解质

     - 等价于：
       1. 中心存在：{}
       2. 方式存在：所是
       3. 注入存在：(自 inject (<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质))
       4. 意指结果：电解质

     

为了方便描述，可以将其写作：在水溶液中或熔融状态下能够电离出离子的某物[确定某物为: 化合物] =: 电解质

- 其也可被视作这样一条构造：化合物[确定特征为: 在水溶液中或熔融状态下能够电离出离子] =: 电解质

- 该构造也可被改写为：化合物[确定特征为: A的构造结果] =: 电解质

  - 构造A：能够电离出离子[确定发生条件为: 在水溶液中或熔融状态下] =: 在水溶液中或熔融状态下能够电离出离子

  - 构造A又可以被改写为：能够电离出离子[确定发生条件为: B构造结果] =: 在水溶液中或熔融状态下能够电离出离子
    - 构造B：在某情况下[确定某情况为: C的构造结果] =: 在水溶液中或熔融状态下
    - 构造C：D的构造结果[确定第二参数为: 熔融状态]  =: 水溶液中或熔融状态
    - 构造D：...或...[确定第一参数为: 水溶液中] =: 水溶液中或...

我们可以把任意的知识点不断递进细分成尽可能小的构造的组合，从而便于机器理解不同存在之间的可能关系。

题干本身引发出一个存在，此时有构造1

3. 注入：文字存在

33. 关于意指=

- 定义 ⇒ 若(F P X) = P'，则可将(F P X)代换为P'。
- 意指是一种操作或事实
- (F P X) = P'作为事实或操作出现，但也同时提供了"F"的定义。
- 可以视为对函数F，当参数1=P，当参数2=X，将(F P X)取值为P。

43. 关于"是"=:

- 定义 ⇒ 若有(F B X) = A，则认为A =: B	
- 推论 ⇒ 若有B = A，则认为A =: B
  - 原理 ⇒ 由于总有(= （F B X) self) = (F B X)，故若有(F B X) = A，则有(= （F B X) self) = A，即A =: B且A =: (F B X)
- 若(F B X) =: A，若无(= A self)外已知的(G A Y) = (F B X)，则意味着必需有(F B X) = A
- 若A := B，且无A =: B,则必有(F C X) = B，且对于该C，无(G C X) = A
- 注意，既有A := C，又有A := B是可能的，这使得在":="层面A有同时表达B或C的能力，由此可形成聚类。

44. 关于意指的完全等价性（若A = B，则B = A）

- 对于A = B，实则总是(F A X) = (F B X)，只是当F、X未明时，认为有全局A = B。
- 注意，除了一些内部逻辑，由外部世界获取的完全等价性大多是一种判断而非必然，A=B，在下一次接收新信息时，完全等价性可能被破坏从而丧失。
- 例如三角形的内角和与180度有完全等价性，但假设有一个幻想的可能世界，这个世界中有能改变某些物理数学规则的神明，这里三角形的内角和是185度而不是180度，这种完全等价性便在这种可能世界中丧失。不同的可能世界靠F、X来区分，若两者F、X分别完全等价，即认为其P都处在同一可能世界下，否则处于不同可能世界中，即对于(F A X)和(F B X)，认为其A、B处于同一可能世界(F ? X)，若总有(F A X) = (F B X),(F B X) = (F A X)，则认为A和B在可能世界(F ? X)中完全等价

4. "⇒"后的M，默认为"self"，即\<F A X>

## 举例

对于“3大于2”这一知识点，可以描述为：

3

2

大于

- ~的左操作数是~3 ⇒ *大于*：~的右操作数是~*2*
- 3 ⇒ *大于*：2 //→: 3 ⇒ *大于*：→: *2*，可以推出~的左操作数是~3 ⇒ *大于*：~的右操作数是~*2*

大于'

- ⇒ 大于
- 左操作数 ⇒ 3
- 右操作数 ⇒ 2

## 转写规则

对于任意构造(<<i (<<c R (F P X)) Q)，都可以转写为6种描述为：

### 1.X为主题

X

- ~F: P ⇒ R: Q //(<<i (<<c R (F P X)) Q)

### 2.Q为主题

Q

- self ⇒ ~R: (F P X)

### 3.R为主题

R

- ~的[A]是~(F P X) ⇒ *R*: ~的[B]是~*Q* // [A] [B]表示R的第一参数和第二参数，即可以有(R '[A] [B]) = [A]

### 4.F为主题

F

- ~的[A]是~(F P X) ⇒ *F*: ~的[B]是~*X*

### 5.P'为主题(即(F P X)为主题)	

 (F P X)

- self ⇒ R: X

### 6.P为主题（若P是非存在数据需要先存在化）

P

- G:  self ⇒ R: Q //其中 (~G self P)  =: (F P X) ， (<<i (<<c R (~G self P) )) Q)
- ~FX的~self ⇒ R: Q

举例：

(<<i (<<c = (自 颜色 苹果)) 红色)，~自 =: 的，~的 =: 自

苹果

- →: 颜色 ⇒ =：红色
- 颜色 ⇒ =：红色

颜色

- ~自苹果的~self ⇒ =：红色
- ~←苹果的~self ⇒ =: 红色 //←表示自

## 通用构造

对于任意一句话S，如果想将对其中某个存在E作为主题，都可以如此构造：

E

- 作为[A]参与到: 'S //'S是S中E所在的部分用[A]表示后的结果
- 'S //直接用[@]在'S中表示[A]

举例：以3为主题，描述3大于2这一知识点

3

- 大于：2 //即 →: self ⇒ 大于：2，(<<i (<<c 大于 3) 2)
- 作为[A]参与到: [A]大于2，(<<i (<<c 作为[A]参与到 3) [A]大于2)
- [@]大于2 //即作为[A]参与到: [A]大于2，(<<i (<<c 作为[A]参与到 3) [A]大于2)

## 方式中置

A

- F: X ⇒ 作为center参与到：R
  - inject ⇒ M
- G: Y ⇒ T: R'

也可简写为：

A

- G: Y ⇒ T: A｢F: X ⇒ R: M｣

进一步简写为：

A

- G: Y ⇒ T: ｢F: X ⇒ R: M｣

若存在｢F P X｣，则

1. 存在'P，能使(F 'P X) = P
2. 存在~F' = ｢~F X P｣
3. 存在X'  = (~F X P) = \<F P X>
4. 存在'X，能使(F 'X P) = X

### 常用推论

1. 若存在(F P X) = P'，则
   1. 存在｢F P' X｣
   2. 存在'X，能使(~F 'X P') = X
   3. 存在X'，能使(~F X P') = X'
   4. P' = (F P X)
   5. P' :=: (F P X)
   6. P'/(F P X) =: P
2. 若存在｢F P X｣，则
   1. 存在'P，能使(F 'P X) = P
   2. 存在~F' = ｢~F X P｣
   3. 存在X'  = (~F X P) = \<F P X>
   4. 存在'X，能使(F 'X P) = X
