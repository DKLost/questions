# 关于本软件中知识点的实现

任意一条知识点都被视作一条构造。

一条构造由四部分组成：中心数据，方式存在，注入存在，构造结果。

一条构造可以理解为对中心P以方式F注入存在X，从而构造一个仍是P的结果P'的过程。

对一个知识点的复习，即被视作再引发一条构造，并判断引发结果是否正确的过程。

通常题干-答案式的知识记录，答案本身便是一种对构造结果的引起，而构造过程，即方式、中心、注入三者则被隐含在了题干当中。

为了方便描述，可以将任意一条构造都可以转化描述为：P'是将中心P的Y确定为X的P。其实质是找到一个方式为.Y=的等价构造。

例如：

问题：在水溶液中或熔融状态下能够电离出离子的化合物是什么？。

答案：电解质。

其本身可被视作这样的一条构造：

1. 中心数据：在水溶液中或熔融状态下能够电离出离子的某物
2. 方式存在：确定某物为
3. 注入存在：化合物
4. 结果存在：电解质

为了方便描述，可以将其写作：在水溶液中或熔融状态下能够电离出离子的某物[确定某物为: 化合物] = 电解质

- 其也可被视作这样一条构造：化合物[确定特征为: 在水溶液中或熔融状态下能够电离出离子] = 电解质

- 该构造也可被改写为：化合物[确定特征为: A的构造结果] = 电解质

  - 构造A：能够电离出离子[确定发生条件为: 在水溶液中或熔融状态下] = 在水溶液中或熔融状态下能够电离出离子

  - 构造A又可以被改写为：能够电离出离子[确定发生条件为: B构造结果] = 在水溶液中或熔融状态下能够电离出离子
    - 构造B：在某情况下[确定某情况为: C的构造结果] = 在水溶液中或熔融状态下
    - 构造C：D的构造结果[确定第二参数为: 熔融状态]  = 水溶液中或熔融状态
    - 构造D：...或...[确定第一参数为: 水溶液中] = 水溶液中或...

我们可以把任意的知识点不断递进细分成尽可能小的构造的组合，从而便于机器理解不同存在之间的可能关系。

为了减轻构造的难度，目前采取分离式设计，即答案本身之时只挂钩到某一构造上，但构造的中心、方式、数据、结果都为空。

# 题干-答案

任何一条题干-答案，都可以视为有3个构造组成：

题干本身引发出一个存在，此时有构造1

从构造1的构造结果，确定其对应的文字形式的存在的过程，此时有构造2

从答案承载的文字数据，确定其对应的文字形式的存在，此时有构造3

构造1：

1. 中心：题干隐含
2. 方式：题干隐含
3. 注入：题干隐含
4. 结果：某存在A

构造2：

1. 中心：空文字存在
2. 方式：确定所指为
3. 注入：A
4. 结果：文字存在B

构造3：

1. 中心：答案承载的文字数据
2. 方式：是
3. 注入：存在
4. 结果：文字存在B（答案承载的文字数据的存在）

# 笔记格式

完整格式：

A

- ~F~X ⇒ R: M //相当于(<<2 (<<1 R (~F X A)) M)	
  - ~G~Y ⇒ T: N //相当于(<<2 (<<1 T (~G Y X')) N)，其中(<<2 (<<1 == X') (~F X A))
    - ...
  - ...
- ...

简写格式:

当省略时，F默认为“的”；X默认为“self”；R默认为“是”；M必须和R一起省略，相当于“是: X”。

A

- ~F~X ⇒ R: M //相当于(<<2 (<<1 R (~F X A)) M)
- X  ⇒ R: M //省略~F~，等价于~的~X ⇒ R: M，相当于(<<2 (<<1 R (确定约束者为 X A)) M)
- ~F~⇒ R: M //省略X，等价于~F~self⇒ R: M，相当于(<<2 (<<1 R (~F self A)) M)
- ~F~X ⇒ M //省略R: ，等价于~F~X ⇒ 是: M，相当于(<<2 (<<1 := (~F X A)) M)
- R: M //省略~F~X ⇒ ，等价于~的~self ⇒ R: M，相当于(<<2 (<<1 R A) M)，该格式不允许有子项
- X ⇒ M //省略~F~ R: ，等价于~的~X ⇒ 是: M，相当于(<<2 (<<1 := (~确定约束者为 X A)) M)
- ~F~X: //省略⇒ R M，等价于~F~X ⇒ 是: X，相当于(<<2 (<<1 := (~F X A)) M)，该格式必须有子项
- X: //省略~F~ ⇒ R M，等价于~的~X ⇒ 是: X，相当于(<<2 (<<1 := (确定约束者为 X A)) M)，该格式必须有子项
- M //省略~F~X ⇒ R:，等价于~的~self ⇒ 是: M，相当于(<<2 (<<1 := A) M)，该格式不允许有子项

1. R: M //等价于~的~第n存在 ⇒ R: M，相当于(<<2 (<<1 R (确定约束者为 (确定序号为 {} 1) A)) M)
2. M //等价于~的~第n存在 ⇒ 是: M，相当于(<<2 (<<1 := (确定约束者为 (确定序号为 {} 1) A)) M)

若M是(~G N A)，则可表示成~G~*N*，其中只用*N*是等价于~的~*N*，即相当于(确定约束者为 N A)

若R是A，则可表示成*R*

"~的~X"，也可写作.X吗，都表示(确定约束者为 X A)
