# 关于本软件中知识点的实现

任意一条知识点都被视作一条构造。

一条构造由四部分组成：中心存在，方式存在（构造过程），注入存在，构造结果（意指结果）。

- 注：由于构造结果实则是完全确定的，即存在一个唯一的直接构造结果，本身是不必要明示的，因而后续描述中的构造结果，大多指向非唯一的意指结果。

一条构造可以理解为对中心P以方式F注入存在X，从而构造一个仍是P的结果P'的过程。

对一个知识点的复习，即被视作再引发一条构造，并判断引发结果是否正确的过程。

通常题干-答案式的知识记录，答案本身便是一种对构造结果的引起，而构造过程，即方式、中心、注入三者则被隐含在了题干当中。

为了方便描述，可以将任意一条构造都可以转化描述为：P'是将中心P的Y确定为X的P。其实质是找到一个方式为.Y=的等价构造。

例如：

问题：在水溶液中或熔融状态下能够电离出离子的化合物是什么？。

答案：电解质。

其本身可被视作这样的一条构造：

1. 中心数据：在水溶液中或熔融状态下能够电离出离子的某物
2. 方式存在：确定某物为
3. 注入存在：化合物
4. 意指结果存在：电解质

- 注意

  1. 对于"在水溶液中或熔融状态下能够电离出离子的化合物是电解质"，(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)，是完整表述

  2. "结果存在"本身是自动生成的一种固定结果，并不具有指向其他存在的能力，这里方便描述，使用的是意指后的结果，使其可以指向其他存在，实则是承载了某种"是"的意指存在。即实质是：

     先有：

     1. 中心存在：{}
     2. 方式存在：所是
     3. 注入存在：(<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物))
     4. 意指结果：(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)

     后有：

     1. 中心存在：inject
     2. 方式存在：自
     3. 注入存在：(<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质)
     4. 意指结果：电解质

     - 等价于：
       1. 中心存在：{}
       2. 方式存在：所是
       3. 注入存在：(自 inject (<<i (<<c 是 在水溶液中或熔融状态下能够电离出离子的化合物) 电解质))
       4. 意指结果：电解质

     

为了方便描述，可以将其写作：在水溶液中或熔融状态下能够电离出离子的某物[确定某物为: 化合物] = 电解质

- 其也可被视作这样一条构造：化合物[确定特征为: 在水溶液中或熔融状态下能够电离出离子] = 电解质

- 该构造也可被改写为：化合物[确定特征为: A的构造结果] = 电解质

  - 构造A：能够电离出离子[确定发生条件为: 在水溶液中或熔融状态下] = 在水溶液中或熔融状态下能够电离出离子

  - 构造A又可以被改写为：能够电离出离子[确定发生条件为: B构造结果] = 在水溶液中或熔融状态下能够电离出离子
    - 构造B：在某情况下[确定某情况为: C的构造结果] = 在水溶液中或熔融状态下
    - 构造C：D的构造结果[确定第二参数为: 熔融状态]  = 水溶液中或熔融状态
    - 构造D：...或...[确定第一参数为: 水溶液中] = 水溶液中或...

我们可以把任意的知识点不断递进细分成尽可能小的构造的组合，从而便于机器理解不同存在之间的可能关系。

# 题干-答案

任何一条题干-答案，都可以视为由3个构造组成：

题干本身引发出一个存在，此时有构造1

从构造1的构造结果，确定其对应的文字形式的过程，此时有构造2

构造1：

1. 中心：题干隐含
2. 方式：题干隐含
3. 注入：题干隐含
4. 结果：某存在A

构造2：

1. 中心：空文字存在
2. 方式：确定形式为
3. 注入：文字存在
4. 结果：A的文字形式

## 构造规则

1. → 表示意指，A → B表示(F P X)意指P'，即可将符号"(F P X)"替换为符号"P'"
2. = 表示"是"，A = B表示A可由B经构造意指得到，即若有(F B X) → A，则认为A = B
3. == 表示双向的"是"
4. === 表示完全等价，即完全相互意指，也可写作<===>
5. {} 表示一个原初"存在"，一个尚无描述的空的存在
6. … 表示原初方式存在，即最原始的方式存在
7. 对于任意存在P，都有P = {}
8. 对于任意方式存在F，都有F = …
9. F === \~(~F)
10. (~… ~ F) → ~F
11. (F P X) 表示(F P X)的直接意指(设为P',P'与(F P X)是完全等价的)
12. (F P X) === (F P X) = P
13. \<F P X> === (~F X P) = X
14. ｢F P X｣ === (<<i (<<c F P) X) = F //<<i 表示绑定inject，<<c表示绑定center
15. (… A B) 表示(…)B的A
16. <… A B> 表示A(…)的B
17. ｢… A B｣ 表示A(…)B
18. <<c === ｢… c (→ center i)｣, (<<c F P) === (… F (→ center P))
19. <<i === ｢… c (→ inject i)｣, (<<c F I) === (… F (→ inject I))
20. 若(F P X) → P',则存在'X，使(~F 'X (→ P P')) === X === (｢~F c (→ P i)｣ 'X P')
21. 若(F P X) → P',则存在'X，使(~F 'X P') → X
22. (F P X) → (F P X)总是可能的
23. 若存在(F P X) → P'，则P' → (F P X)是可能的
24. 总有<… (F B X) self> === (F B X)
25. 总有(→ （F B X) self) === (F B X)
26. 若(F (G P Y) X) === P'',(F P' X) === P''，则必有(G P Y) === P'
27. 若(F P X) → P',则｢F P' X｣
28. 若｢F P X｣, 则有\<F P X>
29. 若｢F P X｣，则存在'P，使(F 'P X) → P
30. 若(F 'P X) → P,(G 'X Y) → X,则(F 'P (G 'X Y)) → P
31. 若(F A X) → A'，若(G B Y) === A，则(F (G B Y) X) → (G (F B X) Y) → A'
32. 若(F A X) === A'，若(G B Y) === A，则(F (G B Y) X) === (G (F B X) Y) === A'
33. 若(F B X) → A，则｢→ (F B X) A｣
34. 若对于任意'P,凡有(F 'P X) → P,总有(F 'P (G 'X Y)) → P,则(G 'X Y) → X
35. 关于意指

- 定义 ⇒ 若(F P X) → P'，则可将(F P X)代换为P'。
- 意指是一种操作或事实
- (F P X) → P'作为事实或操作出现，但也同时提供了"F"的定义。
- 可以视为对函数F，当参数1=P，当参数2=X，将(F P X)取值为P。
- 注意，若既有(F A X) → C，又有(F A X) → B是可能的，这使得不执行意指操作时的(F A X)自身A'有同时表达B或C的能力，由此可形成聚类。
- 若A→B，且无A===B,则必有(F C X) → B，且无(G C X) → A

38. 关于=

- 定义 ⇒ 若有(F B X) → A，则认为A = B	
- 推论 ⇒ 若有B → A，则认为A = B
  - 原理 ⇒ 由于总有(→ （F B X) self) === (F B X)，故若有(F B X) → A，则有(→ （F B X) self) → A，即A = B且A=(F B X)
- 若(F B X) = A，若无(→ A self)外已知的(G A Y) → (F B X)，则意味着必需有(F B X) → A

39. 关于完全等价性

- 注意，除了一些内部逻辑，由外部世界获取的完全等价性大多是一种判断而非必然，A===B，在下一次接收新信息时，完全等价性可能被破坏从而丧失。
- 例如三角形的内角和与180度有完全等价性，但假设有一个幻想的可能世界，这个世界中有能改变某些物理数学规则的神明，这里三角形的内角和是185度而不是180度，这种完全等价性便在这种可能世界中丧失。不同的可能世界靠F、X来区分，若两者F、X分别完全等价，即认为其P都处在同一可能世界下，否则处于不同可能世界中，即对于(F A X)和(F B X)，认为其A、B处于同一可能世界(F ? X)，若总有(F A X) → (F B X),(F B X) → (F A X)，则认为A和B在可能世界(F ? X)中完全等价

### 常用推论

1. 若存在(F P X) → P'，则
   1. 存在｢F P' X｣
   2. 存在'X，能使(~F 'X P') → X
   3. P' → (F P X)
   4. P' == (F P X)
   5. P'/(F P X) = P
2. 若存在｢F P X｣，则
   1. 存在'P，能使(F 'P X) → P
   2. 存在~F' === ｢~F X P｣
   3. 存在X'  === (~F X P) === \<F P X>
   4. 存在'X，能使(F 'X P) → X
